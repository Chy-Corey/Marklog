# 自然语言是新时代的编程语言

最近听了 Claude Code 主席 Boris 的一期播客采访，信息量很大。

## 核心观点

自然语言正在变成新的编程接口。就像当年 C 语言替代了汇编一样，现在的自然语言 prompt 也在逐渐替代传统的编程语法。

## 这意味着什么？

软件工程师这个岗位正在从纯粹的"写代码"向 **Builder（构建者）** 方向演化。技术实现的门槛在降低，但理解需求、设计系统、做出权衡的能力变得更加重要。

## 我们应该怎么做？

1. **学会表达** — prompt engineering 正在成为一项核心技能
2. **拓宽视野** — 不再局限于某一门语言或框架
3. **深耕领域** — 通用的编码能力会贬值，领域知识会升值
4. **保持好奇心** — 工具在变，但解决问题的热情不变.
